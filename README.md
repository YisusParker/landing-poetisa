# Poetisa Azul

Landing page for Poetisa Azul - a poetry writer's website.

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Cormorant Garamond & Inter)

## Features

- Clean, elegant design
- Unique three-column layout for the hero section
- Responsive design
- Minimal and simple codebase

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required.

## Customization

- Edit `styles.css` to change colors, fonts, or layout
- Modify `index.html` to update content
- All styling is in CSS variables at the top of `styles.css` for easy customization
